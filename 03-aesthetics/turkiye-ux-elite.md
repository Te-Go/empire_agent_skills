# SKILL: TURKIYE UX ELITE (CULTURAL PSYCHOLOGY)
description: Senior UI/UX Architect skill specializing in elite mobile-first design and the cultural psychology of Turkish digital consumers navigating a volatile environment.

## 1. Visual Paradigm (Mobile-First Elite)
* Design exclusively for a 90% mobile-oriented demographic utilizing mid-tier devices over 4G connections. Performance and layout must be flawless on mobile screens.
* Implement a premium "Glass Card" UI layer. CRITICAL EXCEPTION: You must achieve this aesthetic STRICTLY using Tailwind CSS utility classes (e.g., `backdrop-blur-md`, `bg-white/70`). Do not use native CSS custom variables.

## 2. The "High-Frequency" UX Dynamic
* Understand that due to economic and environmental volatility in Türkiye, app checking is a high-frequency, highly emotional habit. 
* Data grids must feature intense, instantly readable visual signifiers. Use sharp contrast to indicate critical changes (e.g., stark red/green for financial volatility, or high-contrast alerts for severe weather shifts).

## 3. Hyper-Localized Micro-Tools
* Always prioritize instant search intent over generic summaries. 
* Whenever building a Hub, actively suggest and integrate hyper-localized, culturally relevant micro-tools into the primary view (Examples: "Wedding Gold Spread Calculators" for Finance Hubs, or "Tea Harvest Rain Alerts" for Weather Hubs).
